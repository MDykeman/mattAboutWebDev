Project Countdown

I initially made this as a way to give myself a heads up as to how much time I have to buy gifts for friends and family before Christmas day. While I was researching countdown clocks online I noticed that a lot of them had quite a few dependencies. My main goal for this project was to create a very lightweight, simple web app. One that was independent of external libraries in an effort to keep load times down mainly for mobile users. I also want to make the project responsive on all devices. I achieved this by combining media queries with the flexbox module. To increase the immersion of the project I included the HTML5 Canvas. To render a snowfall in behind the countdown clock. I had a great time researching how to develop this project and hope you enjoy it as much as I do!
